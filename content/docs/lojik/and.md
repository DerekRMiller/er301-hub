## And

A logical `And` gate.

Output goes high when the **unit input** _and_ the **gate** input are greater than zero.

| Input            | Description                        |
| ---------------- | ---------------------------------- |
| **_unit input_** | The left input to the `and` gate.  |
| **gate**         | The right input to the `and` gate. |
